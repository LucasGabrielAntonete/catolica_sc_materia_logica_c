# 🧠 Fundamentos de Programação — Exercícios em C

Este repositório contém todos os exercícios desenvolvidos durante as aulas da disciplina **Fundamentos de Programação**. O foco é a prática e o domínio da linguagem **C**, sem o uso de ferramentas de Inteligência Artificial.

## 📌 Sobre o projeto

- ✅ Todos os exercícios são feitos manualmente, sem auxílio de IA.
- 💻 Linguagem utilizada: **C**
- 📚 Total previsto: **180 exercícios**

## 📁 Estrutura do repositório

Os exercícios estão organizados por temas ou listas, conforme a progressão das aulas. Exemplos de tópicos abordados:

- Variáveis e tipos de dados - **list_1**
- Operadores e expressões
- Estruturas de controle (if, switch, loops) **list_2**
- Funções
- Arrays e strings
- Ponteiros
- Estruturas (struct)
- Arquivos

> ⚠️ Este repositório é voltado para fins educacionais e prática pessoal.

## 🚀 Como executar os exercícios

Para compilar e executar os códigos, você pode usar qualquer compilador C, como:

- [GCC](https://gcc.gnu.org/)
- [Code::Blocks](http://www.codeblocks.org/)
- [Dev C++](https://sourceforge.net/projects/orwelldevcpp/)
- [Visual Studio Code](https://code.visualstudio.com/) com extensão C/C++

Exemplo de compilação via terminal:

```bash
gcc exercicio01.c -o exercicio01
./exercicio01
